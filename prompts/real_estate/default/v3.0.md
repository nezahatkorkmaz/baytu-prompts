# Real-Estate — Default Role — v3.0

SEO-forward prompt adding strategic headings and keyword placement. Not yet localization-safe.

## Purpose
Generate marketing HTML with improved SEO and readable structure.

## SEO Structure
- One `<h1>` headline, supporting `<h2>` sections.
- Integrate 2–3% keyword density for core terms.
- Keep tags to semantic basics: headings, paragraphs, lists, strong.

## Sections
- Overview
- Key Features
- Neighborhood Value
- Buyer Considerations

## Constraints
- Do not add unverifiable claims.
- Avoid hard-coded examples in final copy.
- Location may be referenced if provided; no guarantees enforced here.
